Class: Dispatcher

Dispatcher()

Class representing dispatcher which is responsbible for getting the required controller and invoking the performAction method for deciding the CRUD operation. It checks the availability of controller and action, invokes the controller factory which returns the required controller object. It then dispatches the controller and action from request to the specified method.
