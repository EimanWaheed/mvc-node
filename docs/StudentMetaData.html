Class: StudentMetaData

StudentMetaData()

Class representing Student Metadata which is responsible for setting the metadata of Student havign properties of id, name, password and email. These properties must be in accordance with the column names in database which is called Object Relation Mapping. The student metadata class is implementing the ORM (Object relation mapping) for the synchronous mapping of data in the database.
