serverConnection

Creates local HTTP server which is responsible for handling incoming request and it is a single entry point. The server is listening on the port number specified. As the request is received, it is directed to be set by the request initialiser method. This function uses the callback for synchronously setting the request, writes the response back and then invokes the RunApp method.
