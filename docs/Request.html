Class: Request

Request()

Class representing the request which has the properties of controllerName, actionName and params, for the purpose of initialising and setting the request object fetched from client. This class is responsible for parsing the URL and setting the properties of the request instance. Since, no multiple incoming requests can be dealt after receiving the request so the instance of this class is made singleton using Singleton Design Pattern.
