Class: CRUDModel

CRUDModel()

Class representing CRUD operations which will be responsible for creating, updating, deleting and listing records. These CRUD operations are perfomed for the specific model and thus querybuilder is responsible for generating the query and returning its results back. This class also handles the use of driver instance which will run the query.
