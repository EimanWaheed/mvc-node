Class: TeacherModel

TeacherModel()

Class representing teacher model which has the property of entity having all the metadata in its instance which will be used for the running its own CRUD operations. Teacher Model is extending CrudModel which explicitly inherits all of the CRUD operations thus ensuring the reusability. The property of teacher model will make it easy to access its metadata.
